#### 4.2 经典题目

##### 4.2.1 反转字符串

###### 例题：leetcode 344 - 反转字符串

[344. 反转字符串](https://leetcode.cn/problems/reverse-string/)

- 编写一个函数，其作用是将输入的字符串反转过来。输入字符串以字符数组 `s` 的形式给出。

- 不要给另外的数组分配额外的空间，你必须原地修改输入数组、使用 O(1) 的额外空间解决这一问题。

```cpp
class Solution {
public:
    void reverseString(vector<char>& s) {
        int n=s.size();
        int left=0,right=n-1;
        while(left<right){
            char temp;
            temp=s[left];
            s[left]=s[right];
            s[right]=temp;
            left++;right--; 
        }
    }
};
```

- 一个比较简单的双指针的写法，当然最直接的写法是reverse()。

###### 例题：leetcode 541 - 反转字符串II

[541. 反转字符串 II](https://leetcode.cn/problems/reverse-string-ii/)

- 给定一个字符串 `s` 和一个整数 `k`，从字符串开头算起，每计数至 `2k` 个字符，就反转这 `2k` 字符中的前 `k` 个字符，再重新计数。

  - 如果剩余字符少于 `k` 个，则将剩余字符全部反转。

  - 如果剩余字符小于 `2k` 但大于或等于 `k` 个，则反转前 `k` 个字符，其余字符保持原样。

```cpp
class Solution {
public:
    string reverseStr(string s, int k) {
        int n=s.length();
        int d=(n/k)*k==n ? n/k:n/k+1;
        for(int i=0;i<d;i+=2){
            if((i+1)*k>=n) reverse(s.begin()+i*k,s.end());
            else reverse(s.begin()+i*k,s.begin()+i*k+k);
        }
        return s;
    }
};
```

- 注意reverse的用法，前面不需要加s。

##### 4.2.2 字符串操作

###### 例题：Kamacode 54 - 替换数字

[卡码54.替换数字](https://kamacoder.com/problempage.php?pid=1064)

- 给定一个字符串 s，它包含小写字母和数字字符，请编写一个函数，将字符串中的字母字符保持不变，而将每个数字字符替换为number。 例如，对于输入字符串 "a1b2c3"，函数应该将其转换为 "anumberbnumbercnumber"。

```cpp
#include<iostream>
#include<string>
using namespace std;

int main(){
    string s;cin >> s;
    int i=0,n=s.length();
    while(i<=n-1){
        if(s[i]-'0'<=9 && s[i]-'0'>=0){
            s.erase(i,1);
            s.insert(i,"number");
            i+=5;n+=6;
        }
        i++;
    }
    for(char a:s) cout<<a;
    return 0;
}
```

- 整体比较简单，注意erase和insert的用法，然后注意每次移动指针的步长。

###### 例题：leetcode 151 - 反转字符串中的单词

[151. 反转字符串中的单词](https://leetcode.cn/problems/reverse-words-in-a-string/)

- 给你一个字符串 `s` ，请你反转字符串中 **单词** 的顺序。

- **单词** 是由非空格字符组成的字符串。`s` 中使用至少一个空格将字符串中的 **单词** 分隔开。

- 返回 **单词** 顺序颠倒且 **单词** 之间用单个空格连接的结果字符串。

- **注意：**输入字符串 `s`中可能会存在前导空格、尾随空格或者单词间的多个空格。返回的结果字符串中，单词间应当仅用单个空格分隔，且不包含任何额外的空格。

```cpp
class Solution {
public:
        void remove(string& s) {
        int slow = 0;   
        for (int i = 0; i < s.size(); ++i) { 
            if (s[i] != ' ') { 
                if (slow != 0) s[slow++] = ' '; 
                while (i < s.size() && s[i] != ' ') { 
                    s[slow++] = s[i++];
                }
            }
        }
        s.resize(slow); 
    }

    string reverseWords(string s) {
        remove(s);
        reverse(s.begin(),s.end());
        int slow=0;
        for(int i=0;i<s.length();i++){
            if(s[i]==' '){
                reverse(s.begin()+slow,s.begin()+i);
                slow=i+1;
            }
        }
        reverse(s.begin()+slow,s.end());
        return s;
    }
};
```

- 题目比较困难的点在于：空格可能出现在任何地方，所以我们得先想办法处理空格，使句子顺序正常，然后我们倒置整个句子，再以每个单词为基准倒置回来。
- 对于去除空格的方法，有点类似于第一章的删除元素做法：
  - 我们设置一个慢指针，如果遇到空格就不变，遇到正常元素就插入到慢指针位置，直到最后再遇不到正常元素后，把字符串末尾的空格通过resize的方法全部消去。

##### 4.2.3 KMP算法

###### 例题：leetcode 28 - 找出字符串中第一个匹配项的下标

[28. 找出字符串中第一个匹配项的下标](https://leetcode.cn/problems/find-the-index-of-the-first-occurrence-in-a-string/)

- 给你两个字符串 `haystack` 和 `needle` ，请你在 `haystack` 字符串中找出 `needle` 字符串的第一个匹配项的下标（下标从 0 开始）。如果 `needle` 不是 `haystack` 的一部分，则返回 `-1` 。

```cpp
class Solution {
public:
    vector<int> getnext(string needle){
        int n=needle.size();
        vector<int> next(n);
        int j=-1;
        next[0]=-1;
        for(int i=1;i<n;i++){
            while(needle[j+1]!=needle[i]&&j>=0) j=next[j];
            if(needle[j+1]==needle[i]) j++;
            next[i]=j;
        }
        return next;
   }
    int strStr(string haystack, string needle) {
        vector<int> next = getnext(needle);
        int j=-1;
        for(int i=0;i<haystack.size();i++){
            while(j>=0&&haystack[i]!=needle[j+1]) j=next[j];
            if(haystack[i]==needle[j+1]){
                j++;
            }
            if(j==needle.size()-1) return i-needle.size()+1;
        }
        return -1;
    }   
};
```

- 相当困难的题，居然因为可以用 return haystack.find(needle) 秒掉就被分到了简单。。。

- 题目采用KMP算法解决：
  - KMP的经典思想就是:**当出现字符串不匹配时，可以记录一部分之前已经匹配的文本内容，利用这些信息避免从头再去做匹配。**

- 具体而言就是维护一个next数组来记录：当我匹配needle数组的i项发生不匹配时，next[i]能告诉我回退到needle数组的那一项去重新做匹配：
  - 举个例子：needle="aabaaf"，haystack="aabaabaaf"当我匹配到第二个b时，发生冲突，因为我要求的needle的第六位是f，这个时候一个最快的方法是回退到第三位去判断当前位是不是b，因为之前已经判断过b的前两位都是aa，于是不用再对比了。
- 通过上述例子，我们就明白了，最重要的东西是去生成一个next数组，next数组的意义就是**记录下标i之前（包括i）的字符串中，有多大长度的相同前缀后缀。**
- 问题来到了如何去生成一个next数组？
  - 我们在记录next中统一把数字减去1，于是next[0]为-1（只是一种记录习惯）
  - 从1开始遍历needle，此时注意两个指针：j+1和i，这个需要用来对比前缀和后缀是否相同：
    - 如果相同，则j+1向前移动一步，继续比对（注意记录next[i]=j）
    - 如果不同，说明前缀和后缀对不上了，需要回退让j回退到对的上的next[j]，再做比对（**这里是递归的思想，比较复杂）**

- 最后回到main：
  - 生成next数组，取初始指针j=-1，然后开始遍历haystack：
  - 如果匹配不上j+1处的needle了，回退直至满足或j=-1
  - 若匹配上了，移动指针
  - 若移动到needle末尾了，则查询到子数组了，输出结果。

###### 例题：leetcode 459 - 重复的子字符串

[459. 重复的子字符串](https://leetcode.cn/problems/repeated-substring-pattern/)

- 给定一个非空的字符串 `s` ，检查是否可以通过由它的一个子串重复多次构成。

```cpp
class Solution {
public:
    bool repeatedSubstringPattern(string s) {
        int j=-1;
        int n=s.size();
        vector<int> next(n);
        next[0]=-1;
        for(int i=1;i<n;i++){
            while(j>0&&s[i]!=s[j+1]) j=next[j];
            if(s[i]==s[j+1]) j++;
            next[i]=j;
        }
        int m=next[n-1]+1;
        if(m!=0&&n%(n-m)==0) return true;
        return false;
    }
};
```

- 核心观点：原字符串-最大的相同前缀后缀子字符串是最小重复子字符串，只需要这个小的字符串长度可以被大字符串长度整除，既可以说明其可由一个子串重复多次构成。
